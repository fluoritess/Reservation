所在位置：订单平台设置
订单设置

基本设置：

下单限制： 10分钟内最多允许下笔订单，默认为10笔，限制输入1-20之间的整数

下单时间：：-：之间的时间段允许用户下单

确认收货： 订单发货后天内不确认收货，订单状态自动变更为已确认收货，如果收货时间填写为空，默认为10天

提交订单页：

发票信息：开启关闭

开启后，用户下单时可以填写是否需要发票，以及发票的具体信息。关闭后，用户下单时无法选择发票信息，默认不提供发票。

申请特价：开启关闭

开启后，用户下单时可以针对该订单申请特价，并需填写申请原因。代客下单也可以申请。

自动审核：开启关闭

订单自动审核将影响您所对接的ERP所取到的订单数据。

延迟审核时间：下单 分钟后订单将自动审核

交货日期：开启关闭

开启后，用户下单时可以选择该订单的期望交货日期，以作为店铺的发货时间参考。

退款退货设置

基本设置：

开启退货服务：是否

退货时间限制：订单完成后超过天后，不能进行退货申请
